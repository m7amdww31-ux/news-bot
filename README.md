# 📰 بوت أخبار ديسكورد

بوت يجيب أخبار عربية عامة/عالمية من مصادر RSS موثوقة:
- **أمر يدوي:** `#اخبار` يجيب آخر الأخبار وقت ما تبين
- **نشر تلقائي:** ينشر الأخبار الجديدة في قناة تحددينها كل ١٥ دقيقة

---

## ١) إنشاء البوت في ديسكورد

1. ادخلي https://discord.com/developers/applications
2. اضغطي **New Application** وسمّيه (مثلاً: أخبار زاويتنا)
3. من القائمة الجانبية → **Bot** → اضغطي **Reset Token** وانسخي التوكن (احفظيه، بتحتاجينه)
4. تحت **Privileged Gateway Intents** فعّلي:
   - ✅ **MESSAGE CONTENT INTENT** (ضروري عشان تشتغل الأوامر)
5. من **OAuth2 → URL Generator** اختاري:
   - Scopes: `bot`
   - Bot Permissions: `Send Messages`, `Embed Links`, `Read Message History`
6. انسخي الرابط اللي يطلع، افتحيه، واضيفي البوت لسيرفرك.

---

## ٢) النشر على Railway (نفس طريقة بوت الموسيقى)

1. ارفعي ملفات هذا المجلد لمستودع GitHub جديد.
2. في Railway → **New Project → Deploy from GitHub repo** → اختاري المستودع.
3. روحي تبويب **Variables** وضيفي:

| المتغيّر | القيمة |
|---|---|
| `DISCORD_TOKEN` | التوكن اللي نسختيه |
| `NEWS_INTERVAL_MINUTES` | `15` (اختياري - كل كم دقيقة ينشر) |
| `NEWS_CHANNEL_ID` | (اختياري) ID القناة للنشر التلقائي |

4. Railway بيبني المشروع تلقائياً ويشغّل البوت.

> 💡 RSS ما ينحظر على Railway مثل ما صار مع يوتيوب، فالأخبار بتشتغل عادي.

---

## ٣) الأوامر داخل ديسكورد

| الأمر | الوظيفة |
|---|---|
| `#اخبار` | يجيب آخر ٥ أخبار |
| `#اخبار 3` | يجيب العدد اللي تبينه (١ إلى ١٠) |
| `#قناة_الاخبار` | يخلي القناة الحالية قناة النشر التلقائي (للمشرفين) |
| `#ايقاف_الاخبار` | يوقف النشر التلقائي (للمشرفين) |
| `#مصادر` | يعرض مصادر الأخبار |
| `#مساعدة` | يعرض كل الأوامر |

**أسهل طريقة لتشغيل النشر التلقائي:** اكتبي `#قناة_الاخبار` في القناة اللي تبين الأخبار تنزل فيها، وخلاص.

---

## ٤) تعديل المصادر

افتحي `bot.py` وروحي لقسم `FEEDS`. كل سطر مصدر:

```python
FEEDS = {
    "الجزيرة": "https://www.aljazeera.net/xml/rss/all.xml",
    "BBC عربي": "https://feeds.bbci.co.uk/arabic/rss.xml",
    ...
}
```

- تبين تضيفين مصدر؟ زيدي سطر جديد بنفس الشكل: `"الاسم": "رابط الـRSS"`.
- لو مصدر ما اشتغل، البوت يتخطاه تلقائياً وما يوقف.

---

## ملاحظات

- البوت يحفظ الأخبار المنشورة في ملف `data.json` عشان ما يكرّرها. هذا الملف يتصفّر لو أعدتِ النشر على Railway (طبيعي).
- أول ما يشتغل البوت يعلّم الأخبار الحالية كـ"مقروءة" عشان ما يصير سبام، وبعدها ينشر الجديد فقط.
